FAQs - Perguntas Frequentes

1. O que é o sistema Freelancer?

O Freelancer é uma plataforma que conecta profissionais a empresas que estão buscando talentos para realizar projetos específicos ou preencher vagas temporárias.

2. Como faço para me cadastrar no sistema?

Você pode se cadastrar clicando no botão "Cadastre-se" na página inicial. Preencha o formulário com suas informações e finalize o cadastro.

3. Como enviar meu currículo para uma vaga?

Na página de vagas disponíveis, clique no botão "Enviar Currículo" ao lado da vaga desejada. Você receberá uma confirmação de que a candidatura foi realizada com sucesso.

4. É necessário pagar para usar o sistema?

Não, o uso do sistema Freelancer é totalmente gratuito tanto para candidatos quanto para empresas.

5. Como posso cadastrar uma nova vaga?

Se você for uma empresa, pode acessar a página "Cadastrar Oportunidades" no menu principal, preencher os dados da vaga e salvá-la no sistema.

6. Onde posso visualizar os currículos enviados?

As empresas podem acessar a página "Gerenciar Currículos" para visualizar todos os currículos recebidos e gerenciá-los.

7. Como faço para editar ou excluir meu currículo?

No momento, a funcionalidade de edição de currículos não está disponível. Porém, você pode excluir e cadastrar novamente um novo currículo.

8. Como garantir que minha candidatura foi enviada?

Após clicar no botão "Enviar Currículo", você verá uma mensagem de confirmação indicando que sua candidatura foi realizada com sucesso.

9. Que tipos de vagas posso encontrar no sistema?

No sistema Freelancer, você pode encontrar vagas como CLT, PJ, Estágio e Freelancer para diferentes áreas de atuação.

10. Como posso entrar em contato com o suporte?

Se precisar de ajuda, entre em contato pelo e-mail suporte@freelancer.com ou utilize o formulário de contato disponível no site.
